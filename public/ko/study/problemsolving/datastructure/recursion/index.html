📜RESURSION(재귀) 2022-09~2022-12

  • **재귀(Recursion)**는 함수가 자기 자신을 호출하는 프로그래밍 기법을 말합니다. 재귀는 주어진 문제를 더 작은 하위 문제로 나누어 반복적으로 해결할 수 있는 상황에서 매우 유용합니다. 이를 통해 복잡한 문제를 간단한 형태로 표현할 수 있습니다. 재귀는 문제를 풀 때 두 가지 주요 요소를 필요로 합니다: **기저 조건(termination/base condition)**과 재귀적 호출(recursive call).

  • 재귀 함수의 구조: 재귀 함수는 일반적으로 다음과 같은 구조를 가집니다:

    • 기저 조건: 재귀가 멈추는 조건을 의미합니다. 기저 조건이 없다면 함수는 무한히 자신을 호출하게 되어 프로그램이 끝나지 않으므로 필수적인 부분입니다.
    • 재귀적 호출: 자기 자신을 호출하는 부분입니다. 이때 매번 호출할 때마다 문제의 크기를 점점 더 작게 만들어 기저 조건에 도달하도록 설계해야 합니다.

  • 개념

    //재귀 Recursion
void func1(int N)
{
	if (N == 0) //Base condition
		return;
	func1(N--);
}

int func2(int N)
{
	if (N == 0) return 0;
	return N + func2(N - 1);
}

int fibo(int n)
{
	if (n <= 1)	return 1;
	return fibo(n - 1) + fibo(n - 2);
	//이 재귀는 시간복잡도가 지수함수 꼴로 터짐
	//왜냐? 이미 연산한 것을 반복함 향후 DP로 해결 가능한 부분도 존재
}

int main()
{
	int n;
	cin >> n;
	func1(n);
	func2(n);
	cout << n;
	return 0;
}

	int half = 1 << (n - 1); //비트 연산자
	if (r < half && c < half) return func(n - 1, r, c); //1번 사각형인 경우
	if (r < half && c >= half) return half * half + func(n - 1, r, c - half); //2번 사각형인 경우
	if (r >= half && c < half) return 2 * half * half + func(n - 1, r - half, c);//3번 사각형인 경우
	return 3 * half * half + func(n - 1, r - half, c - half);
}

#include <iostream> 
using namespace std;
/*
* 1. n-1개의 원판을 기둥1 -> 기둥2
* 2. n번 원판을 기둥1 -> 기둥3
* 3. n-1개의 원판을 기둥2 -> 기둥3
* 
* 재귀식
* n-1개의 원판을 기둥 a -> 기둥 6 - a - b
* n번 원팔을 기둥a -> 기둥b
* n-1개의 원판을 기둥6-a-b -> b
*/
void func1(int a, int b, int n)
{
	if (n == 1) //Base Condition
	{
		cout << a << ' ' << b << '\n';
		return;
	}
	func1(a, 6 - a - b, n - 1);
	cout << a << " " << b << '\n';
	func1(6 - a - b, b, n - 1);
}
int main()
{
	int n;
	cin >> n;
	cout << (1 << n) - 1 << '\n';
	func1(1, 3, n);
}
